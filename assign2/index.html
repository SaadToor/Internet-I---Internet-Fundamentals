HTML5 & CSS Normalize

Hello, and welcome to my HTML5 and CSS website, for assignment 2. Here I will explain why HTML5 is becoming so popular, and the use of Normalize.css.

HTML5 is becoming popular because of all the features that can be done easily. For example, Web Workers: Certain web applications use heavy scripts to perform functions, Multimedia Support in HTML5 such as video and audio, and HTML5 now allows you to add valid data attributes, which helps in storing the data without affecting the web page UI.

The use of Normalize.css is to provide the use of CSS codes to be compatible, and be able to show on every browser without any issues. Normalize.css also maintains the HTML5 codes to stay intact, rather than scatter around the place when opened with differect browsers.
